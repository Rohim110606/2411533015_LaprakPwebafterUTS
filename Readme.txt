----------------------------------------
1. DESKRIPSI
----------------------------------------
Website ini adalah portofolio pribadi responsif yang dibangun menggunakan HTML5 dan kerangka kerja (framework) CSS Bootstrap 5. Tujuannya adalah untuk menampilkan informasi pribadi, pengalaman, kemampuan (skills), dan proyek-proyek (portfolio) yang telah diselesaikan.

Desain menggunakan palet warna Navy Blue (Biru Tua) dan Light Blue (Biru Terang) untuk kesan profesional.

----------------------------------------
2. STRUKTUR FILE
----------------------------------------
Proyek ini terdiri dari file-file utama berikut:

- index.html: Halaman Utama (Home).
- about.html: Halaman Tentang Saya (About Me).
- portfolio.html: Halaman Portofolio/Proyek.
- contact.html: Halaman Kontak (Contact Me).
- style.css: File utama untuk semua styling kustom (custom CSS).

----------------------------------------
3. FITUR UTAMA (BERDASARKAN HALAMAN)
----------------------------------------

A. SEMUA HALAMAN
- Navigasi (Navbar): Fixed/Sticky, responsive, dan dilengkapi Search Bar.
- Header Halaman: Semua heading utama (About Me, My Portfolio, Contact Me) menggunakan ukuran font SANGAT BESAR dan TEBAL (display-1/6rem).
- Footer: Hak cipta dan tautan ke media sosial.

B. ABOUT ME (about.html)
- Personal Info: Teks perkenalan dengan gambar profil (placeholder).
- Timeline: Menampilkan riwayat Pendidikan (School Name, Year) dan Pengalaman/Pencapaian.
- Skills: Daftar kemampuan/tools.

C. PORTFOLIO (portfolio.html)
- Portfolio Grid: Tampilan minimal 6 proyek dalam layout Card (termasuk image, project name, short description, dan badges untuk kategori).
- Filter/Tabs: Proyek dapat difilter berdasarkan kategori (Semua, Web Development, Data Analysis, Project Management).
- Modal Detail: Setiap Card dapat diklik untuk membuka Modal Window yang menampilkan detail lengkap proyek dan Image Carousel.

D. CONTACT ME (contact.html)
- Contact Form: Formulir kontak lengkap (Name, Email, Phone, Subject, Message, Terms Checkbox).
- Form Validation: Dilengkapi client-side validation dan feedback (Alert Success/Error).
- Contact Info: Menyediakan Address, Phone Number, Email, dan Social Media Links.
- Google Maps: Menyematkan peta lokasi.

----------------------------------------
4. DEPENDENSI / TEKNOLOGI YANG DIGUNAKAN
----------------------------------------
1. HTML5
2. CSS3 (Custom Styling)
3. Bootstrap 5.3 (Kerangka kerja CSS utama)
4. Font Awesome 6.5.2 (Untuk ikon, seperti sosial media dan navigasi)
5. JavaScript (Vanilla JS untuk form validation di contact.html dan fungsi Bootstrap)

----------------------------------------
5. CARA PENGGUNAAN
----------------------------------------
1. Pastikan file HTML, CSS, dan folder gambar berada dalam direktori yang sama.
2. Buka salah satu file HTML (disarankan: index.html) di browser modern apa pun (Chrome, Firefox, Edge).
